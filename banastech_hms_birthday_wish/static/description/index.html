Birthday Wish

This module can be useful for any company or user to give birthday greetings to their customers,employees,etc through email for that we have added birthdate field in res partner.

We have Designed template for that and if you want you can design your own mail template.

Set Your Birthday Wish Template

In general setting users can set email template based on their interest or can create new template and set here, Seleted template will be sent to the birthday guy.

This is not limited to mail only if you want to get all birthday notifications you can join Birthday notification group.

Birthday Wishes Group

Interested persons can get all birthday notifications by joining this group.

If Any company want to call also to their special customers than they can do call based on notifications.
